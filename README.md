# waterFilter
control waterfilter system:
  - STM32F401CCU6:
      + operating the system.
      + leds.
      + read DFrobot water pressure sensor.
      + pump.
  - Arduino Leonardo:
      + read 2 sensor YF401s.
      + read high pressure switch through a voltage tranformer.
      + send signal to STM32.
